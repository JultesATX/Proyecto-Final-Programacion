# 🎓 Sistema de Control de Cursos y Estudiantes

Este es el proyecto final del curso de programacion 1. Implementa un sistema CRUD que gestiona cursos académicos y los estudiantes inscritos en ellos, utilizando principios avanzados de Programación Orientada a Objetos utilizando de base el framework EloquentORM.

---

## 🚀 Características Principales

* **Implementación POO**: Uso de clases, herencia y relaciones para modelar el sistema académico de manera facil de manejar.
* **EloquentORM Híbrido**: Utiliza una capa de persistencia flexible que simplifica las consultas complejas y proporciona un ORM ligero sobre MySQL.
* **Base de Datos Relacional**: Utiliza MySQL para un almacenamiento eficiente de datos.
* **Interfaz por Consola**: Presenta un diseño interactivo basado en menus estilo terminal.

## 🛠️ Requisitos del Sistema

Para compilar y ejecutar este proyecto, es necesario:

* Un compilador de C++ (Por ejemplo, `g++`).
* Servidor MySQL instalado y corriendo.
* Librería de desarrollo de MySQL para C/C++.

## 💾 Instalación y Configuración

### 1. Preparar la Base de Datos

* Ejecutar el script SQL incluido para crear la base de datos y las tablas necesarias
* Utilizar las mismas credenciales del codigo main.cpp que por default son user: root password: admin

### 2. Conectar con Visual Studio Code
* Asegurese de que las rutas de inclusion en su archivo c_cpp_properties.json apunten a la carpeta include de tu instalación de MySQL.

### 3. Ejecutar
* Si los pasos anteriores se hicieron de manera correcta puede proceder con la ejecuccion y compilacion del codigo.
* Si todo salio bien vera un menu estructurado con las opciones de Gestionar alumnos y cursos disponibles.
