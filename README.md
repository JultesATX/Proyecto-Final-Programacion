# рҹҺ“ Sistema de Control de Cursos y Estudiantes

Este es el proyecto final del curso de programacion 1. Implementa un sistema CRUD que gestiona cursos acadГ©micos y los estudiantes inscritos en ellos, utilizando principios avanzados de ProgramaciГіn Orientada a Objetos utilizando de base el framework EloquentORM.

---

## рҹҡҖ CaracterГӯsticas Principales

* **ImplementaciГіn POO**: Uso de clases, herencia y relaciones para modelar el sistema acadГ©mico de manera facil de manejar.
* **EloquentORM HГӯbrido**: Utiliza una capa de persistencia flexible que simplifica las consultas complejas y proporciona un ORM ligero sobre MySQL.
* **Base de Datos Relacional**: Utiliza MySQL para un almacenamiento eficiente de datos.
* **Interfaz por Consola**: Presenta un diseГұo interactivo basado en menus estilo terminal.

## рҹӣ пёҸ Requisitos del Sistema

Para compilar y ejecutar este proyecto, es necesario:

* Un compilador de C++ (Por ejemplo, `g++`).
* Servidor MySQL instalado y corriendo.
* LibrerГӯa de desarrollo de MySQL para C/C++.

## рҹ’ҫ InstalaciГіn y ConfiguraciГіn

### 1. Preparar la Base de Datos

* Ejecutar el script SQL incluido para crear la base de datos y las tablas necesarias
* Utilizar las mismas credenciales del codigo main.cpp que por default son user: root password: admin

### 2. Conectar con Visual Studio Code
* Asegurese de que las rutas de inclusion en su archivo c_cpp_properties.json apunten a la carpeta include de tu instalaciГіn de MySQL.

### 3. Ejecutar
* Si los pasos anteriores se hicieron de manera correcta puede proceder con la ejecuccion y compilacion del codigo.
* Si todo salio bien vera un menu estructurado con las opciones de Gestionar alumnos y cursos disponibles.
